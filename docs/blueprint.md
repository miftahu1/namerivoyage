# **App Name**: Nameri Voyage

## Core Features:

- Public Trip Information Display: A responsive public landing page displaying a hero section, detailed trip information (location, dates, duration, organizer), itinerary, important guidelines, and a list of registered participants.
- Student Registration Form: A comprehensive form for students to register for the trip, collecting full name, class/section, phone number, guardian contact, and medical conditions, with a success confirmation message upon submission.
- Admin User Authentication: A secure, simple login system for administrators (teachers) to access the protected management panel using a username and password.
- Registered Students Management: An admin interface to view, search, filter, approve, edit, or delete student registrations, alongside quick statistics on total participants.
- Trip Content Editor: An administrative tool to easily update trip details (dates, location, notes), modify the daily itinerary, and post announcements visible on the public student page.
- AI Itinerary & Packing List Tool: A generative AI tool to assist administrators in drafting engaging descriptions for itinerary items and suggesting relevant packing list items based on the trip's location and duration.

## Style Guidelines:

- Primary color: A vibrant, clear blue (#228DC3), evoking the refreshing tones of a river, ensuring good contrast on a light background.
- Background color: A very light, desaturated blue (#ECF4F7), providing a clean and open feel, subtly echoing the primary hue while maintaining a modern, minimal aesthetic.
- Accent color: A soothing, bright teal green (#4CBFA6), suggestive of lush nature and vibrancy, used for interactive elements and highlights to provide visual interest and prompt action.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, suitable for both short headlines and longer blocks of trip information, reinforcing the minimalist and objective design.
- Use minimalist, outline-style icons across the site, with specific nature-related motifs for location and activities (e.g., mountains, river) to complement the visual theme.
- A card-based layout structure with soft shadows for clear separation of content sections, ensuring a responsive design that adapts seamlessly to various screen sizes. Ample whitespace will contribute to a clean and easy-to-digest user interface.
- Implement subtle, smooth transitions and hover effects on interactive elements like buttons and cards, creating an engaging yet non-distracting user experience. Gentle loading animations for dynamic content will enhance the fluidity of the interface.
